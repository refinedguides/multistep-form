# Multi-step Form

Guide users through a process with a step-by-step form wizard.

### Features Covered

- [x] Dynamic step indicator
- [x] Progress steps
- [x] Navigates steps
- [x] Validate steps
- [x] Manage tab/focus
- [x] Form submission
